# FrutasDoSeuZe

Este é um projeto de consola em C# que demonstra uma aplicação simples para gerir frutas.

## Tecnologias Utilizadas

*   C#
*   .NET
*   Entity Framework Core
*   SQLite

## Arquitetura

O projeto segue uma arquitetura em camadas, com as seguintes pastas:

*   `Models/`: Contém as classes de modelo.
*   `Repositories/`: Contém os repositórios para acesso a dados.
*   `Services/`: Contém a lógica de negócio.
*   `Exceptions/`: Contém as exceções personalizadas.

## Como Executar

1.  Clone este repositório.
2.  Abra a solução no Visual Studio ou no VS Code.
3.  Execute o projeto.
